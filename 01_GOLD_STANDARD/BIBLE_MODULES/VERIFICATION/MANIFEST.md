## PURPOSE

This module contains universal standards for verification, testing, and quality assurance that apply to all software development regardless of project or technology.

---

## SCOPE

### What This Module Covers

- Testing philosophy and doctrine
- Test types and purposes
- Coverage requirements
- Test design patterns
- Evidence standards
- Verification layers
- Failure protocols

### What This Module Does NOT Cover

- Project-specific test cases (see PROJECT_CHRONICLES)
- Tool-specific syntax (reference tool docs)
- Test data (project-specific)
