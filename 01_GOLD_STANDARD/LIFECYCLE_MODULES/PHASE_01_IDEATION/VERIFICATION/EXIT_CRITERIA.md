# PHASE 01: IDEATION
## Exit Criteria

**Purpose:** Conditions that MUST be satisfied before proceeding to Phase 2

---

## MANDATORY DELIVERABLES

- [ ] **Idea Statement Exists**
  - Written down
  - Describes what will be built
  - Understandable by someone with no context

- [ ] **Problem Statement Exists**
  - Clearly articulated
  - Describes current situation
  - Explains why it's a problem

- [ ] **Success Vision Exists**
  - Describes end state
  - Tangible and measurable (even loosely)
  - Focused on user value

- [ ] **Constraints Documented**
  - Known constraints listed
  - (Empty list is acceptable if no known constraints)

---

## QUALITY CHECKS

- [ ] **Clarity Test**
  - Can be explained in 30 seconds
  - No jargon or assumed context

- [ ] **Problem-Value Alignment**
  - The success vision addresses the problem

- [ ] **Principal Confirmation**
  - "This captures what I want to explore"
