## How to Save New Sessions

When you want to save a Q&A exchange, simply say:
- "Save this"
- "Add this to the knowledge base"
- "Preserve this Q&A"

The AI will:
1. Create a new file with the next sequential ID
2. Include date/time stamp
3. Assign a descriptive topic name
4. Update this index
