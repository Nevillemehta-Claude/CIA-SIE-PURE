Key Principle: Data Repository Model
CIA-SIE is a data repository, not an intelligence engine. It stores signals, exposes contradictions, and generates descriptive narratives. It never makes trading decisions, resolves conflicts, or recommends actions. The human user retains all decision-making authority.

CONSTITUTIONAL CONSTRAINTS ON DATA MODEL

  • CHARTS table: NO weight column - all charts have equal standing
  • SIGNALS table: NO confidence or strength columns - raw data only
  • No aggregation tables: No overall_score, net_direction, or consensus fields anywhere

Prohibited Data Patterns
Prohibited Field Why Prohibited Alternative
weight Enables aggregation Display all charts equally
score Implies judgment Display raw direction only
confidence Implies reliability ranking Display signal type only
recommendation Prescriptive action Display description only
priority Implies hierarchy Display in creation order
rank Implies superiority Display alphabetically
overall_direction Aggregates signals Show each signal separately
net_signal Resolves contradictions Show contradictions explicitly
